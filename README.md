# Deep Learning Project

Este proyecto implementa un modelo de Deep Learning para procesar, analizar y clasificar datos usando redes neuronales convolucionales y capas LSTM. Incluye todo el flujo completo: carga de datos, limpieza, entrenamiento y pruebas del modelo.

---

## Características principales

- Procesamiento y preparación de datos con Python.
- Entrenamiento de modelos usando TensorFlow y Keras.
- Arquitectura CNN + LSTM para secuencias e imágenes.
- Scripts modulares para cargar, analizar y visualizar datos.
- Generación de métricas y gráficos de desempeño.
- Guardado y carga de modelos `.keras`.
